### Car App (Backend) using TypeScript


``` bash
mkdir car-management-backend && cd car-management-backend
npm init -y
npm install express mongoose bcryptjs jsonwebtoken dotenv cors
npm install -D typescript ts-node-dev @types/express @types/bcryptjs @types/jsonwebtoken @types/node
npx tsc --init


Eslint setup
npm install --save-dev eslint @typescript-eslint/parser @typescript-eslint/eslint-plugin

npx eslint --init

```
